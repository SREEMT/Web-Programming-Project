The goal of EnrgTrack is to provide a centralized platform where users interested in the energy market can monitor stock performance, organize watchlists, and engage with other users interested in energy-sector investing.

### Features

- View stock data related to the U.S. energy market
- User accounts for a personalized experience
- Custom watchlists to track selected energy stocks
- Community interaction through discussion forums
- Responsive and user-friendly interface

### Technologies Used

- HTML
- CSS
- Bootstrap (for responsive design)
- React
- Stock Market API (for real-time or simulated data)
- SQL-based database (planned for data storage and user management)
